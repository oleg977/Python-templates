# Работа с GitHub Desktop и PyCharm

## GitHub Desktop
1. Установите GitHub Desktop.
2. Клонируйте репозиторий через интерфейс.
3. Добавляйте файлы, создавайте коммиты и отправляйте изменения.

## PyCharm
1. Откройте проект в PyCharm.
2. Настройте Git в `Settings > Version Control`.
3. Используйте встроенный Git-клиент для добавления файлов, создания коммитов и отправки изменений.
