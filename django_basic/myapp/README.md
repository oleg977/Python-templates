# Приложение 'myapp'

## 1. Назначение

Это базовое приложение проекта `django_basic`. Оно служит для размещения основных, общих страниц, которые не требуют сложной логики (например, главная страница, "О нас", "Контакты" и т.д.).

## 2. Используемые модели (myapp/models.py)

| Модель | Описание |
| :--- | :--- |
| *Нет* | В этом приложении модели отсутствуют, оно ориентировано только на представления. |
| *Пример* | `Item` - для хранения списка чего-либо (если бы это было приложение для списка). |

## 3. URL-маршруты (myapp/urls.py)

| Маршрут | Имя (name) | Представление | Назначение |
| :--- | :--- | :--- | :--- |
| `/` | `index` | `views.index` | Главная страница сайта. |
| `/about/` | `about` | `views.about` | Статическая страница с информацией. |

## 4. Зависимости и настройки

Для активации, добавьте `'myapp'` в `INSTALLED_APPS` в главном файле `settings.py`.
